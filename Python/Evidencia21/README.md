# Consulta de Brechas de Seguridad

## Descripción
Este proyecto permite consultar si un correo electrónico ha sido
comprometido en brechas de seguridad usando una API.

## Funcionalidades
- Lectura segura de API key
- Consulta de brechas por correo
- Registro de actividad con logging
- Generación de reporte en CSV
